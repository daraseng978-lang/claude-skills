## Niche Thesis

- **Decision-driver:** Dementia-specific care capability (wandering prevention, cognitive therapy programs, secured memory units)
- **TAM:** 6.7M Americans with Alzheimer's/dementia, growing ~7%/yr. Memory care segment is ~$25B/yr.
- **Tailwind:** Aging baby boomers; 1 in 9 adults 65+ has Alzheimer's.
- **Lead value:** $500–2,000 (highest in portfolio). Families pay $5K–$10K/mo for memory care.
- **AI-search resilience:** Decision-stage, emotional, high-risk — families do deep research regardless of LLM search
- **Incumbent:** APlaceForMom dominates general senior living (824K/mo visits) but is not dementia-specialized — our wedge

## Filter Schema

- Memory care license (state-specific)
- Secured memory unit (yes/no)
- Dementia-trained staff (certifications)
- Wandering prevention features
- Behavioral management programs
- Cognitive therapy (music, art, reminiscence)
- Price per month (transparency is a real wedge)
- Medicare/Medicaid acceptance
- Respite care availability

## YMYL Consideration

Senior care = high YMYL (Your Money Your Life) sensitivity. Google/LLMs rank this conservatively. Mitigations:
- Cite authoritative sources (Alzheimer's Association, state licensing boards)
- Include disclosures on every page
- Build author pages for any editorial content
- Avoid any medical claims
